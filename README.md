# Random Password Generator

**Автор:** Петр Блинов

## Описание
Это простое GUI-приложение для генерации случайных паролей с возможностью настройки длины и типа символов. В приложении реализована история с сохранением в JSON и управление через удобный интерфейс.

## Использование
- Установите Python 3.
- Запустите скрипт.
- Настройте параметры: длину пароля, типы символов.
- Нажмите "Генерировать" — новый пароль появится в поле и сохранится в истории.
- История отображается ниже, с возможностью сохранения и загрузки.
